# Guide Installation 

1. Installation Python (3.7+)
2. Type ``pip install --user -U conan`` dans le terminal
   1. Unix: ajouter conan au PATH: ``source ~/.profile``
3. Tester la commande: ``conan``
    1. conan --version
4. conan profile detect --force
5. conan profile path default
